# 选修二 第一章 数列

## 等差数列

**基本公式**

$a_n=\begin{cases}a_1+\left(n-1\right)\cdotp d\\a_m+\left(n-m\right)\cdotp d&\end{cases}$



$S_n=\begin{cases}\frac{\left(a_1+a_n\right)\cdot n}{2}\\\\a_1n+\frac{n\left(n-1\right)}{2}\cdot d&\end{cases}$

**二级结论**

1. 已知 $\{a_n\}$ 为等差数列，若 $m+n=p+q$ ，则$a_{m}+a_{n}=a_{p}+a_{q}$

2. 若 $\{a_n\}$ 为等差数列，则 $S_n，S_{2n}-S_n，S_{3n}-S_{2n}，...$ 也为等差数列



## 等比数列

**基本公式**

$a_n=\begin{cases}a_1\cdot q^{n-1}\\a_m\cdot q^{n-m}&\end{cases}$



$S_n=\begin{cases}\dfrac{a_1\left(1-q^n\right)}{1-q},&q\neq1\\[2ex]n\cdot a_1,&q=1\end{cases}$

**二级结论**

1. 已知 $\{a_n\}$ 为等比数列，若 $m+n=p+q$ ，则$a_{m}\cdot a_{n}=a_{p}\cdot a_{q}$

2. 若 $\{a_n\}$ 为等比数列，则 $S_n，S_{2n}-S_n，S_{3n}-S_{2n}，...$ 也为等比数列
