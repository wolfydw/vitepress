# 必修二 第二章 平面向量及其应用

## 相关定义

- 既有大小又有方向的量统称为**向量**（如位移、速度、力等）
- 只有大小没有方向的量称为**数量**（如年龄、长度、质量、面积、体积等）
- 具有方向和长度的线段称为**有向线段**，记作 $\overrightarrow{AB}$；线段AB的长度称为有向线段$\overrightarrow{AB}$的长度，记作$|\overrightarrow{AB}|$
- 向量可以用有向线段表示，也可以用 $\vec{a}，\vec{b}，\vec{c}，⋯$ 来表示
- 向量 $\vec a$ 的大小，记作 $|\vec a|$ ，又称作**向量的模**
- 模等于1个单位长度的向量称为**单位向量**。



## 向量的基本关系

- **相等向量**是指它们的长度相等且方向相同，向量 $\vec a$ 与 $\vec b$ 相等，记作 $\vec a=\vec b$
- 若两个非零向量 $\vec a$ 与 $\vec b$ 的方向相同或相反，则称这两个向量为**共线向量**或**平行向量**，也称这两个向量共线或平行，记作 $\vec a\parallel\vec b$
- 若两个向量的长度相等、方向相反，则称它们互为**相反向量**，相反向量是共线向量。若其中一个向量为 $a$，则它的相反向量记作 $-a$



## 向量的夹角

- 已知两个非零向量 $\vec a$ 和 $\vec b$ ，在平面内选一点O（自定义起始点），作 $\overrightarrow{OA}=\vec a,\overrightarrow{OB}=\vec b$ ，则 $\theta=\angle AOB(0^\circ \leqslant\theta\leqslant180^{\circ})$ 称为**向量 $a$ 与 $b$ 的夹角**
- 当 $\theta = 0^\circ$ 时， $\vec a$ 与 $\vec b$ 同向；当 $\theta = 180^\circ$ 时， $\vec a$ 与 $\vec b$ 反向；当 $\theta = 90^\circ$ 时， $\vec a$ 与 $\vec b$ 垂直，记作 $\vec a \perp \vec b$



### 零向量

- 长度为 0 的向量称为**零向量**，记作 $\vec 0$ ，零向量的相反向量仍是零向量；
- 任何方向都可以作为零向量的方向。
  - 规定**零向量与任一向量共线**，即对于任意的向量 $a$，都有 $0 \parallel a$，
  - 规定**零向量与任一向量垂直**，即对于任意的向量 $\vec a$，都有 $0 \perp \vec a$



## 向量的加法、减法和数乘的综合运算

- 向量加法中，若起点重合，适合 `平行四边形法则` ；若首尾顺次相连，适合 `多边形法则`
- 向量减法中，若起点重合，指向被减数；例如 $\overrightarrow{OB}-\overrightarrow{OA}=\overrightarrow{AB}$
- 向量减法中，若终点重合，指向减数；例如 $\overrightarrow{AC}-\overrightarrow{BC}=\overrightarrow{AB}$
- 向量的综合计算满足线性运算

例题：

- 三角形重心向三角形三个顶点的向量相加等于0
- 线段 AB 中点 P 的向量表达式：$\overrightarrow{OP}=\frac{1}{2}(\overrightarrow{OA}+\overrightarrow{OB})$



## 向量共线的关系

**共线（平行）向量基本定理**：给定一个非零向量 ，则对于任意向量 $a$，$a//b$ 的充要条件是存在唯一一个实数 $\lambda$，使 $a=\lambda b$



## 平面向量的分解

问题：对于平面内给定的两个不共线向量 $\vec{e_1}，\vec{e_2}$，任一向量 $\vec a$ 是否都可以用形如 $\lambda_1\vec{e_1}+\lambda_2\vec{e_2}$ 的形式表示呢？

答案：**可以**，将向量 $\vec a$ 分别向向量 $\vec{e_1}，\vec{e_2}$ 所在直线的方向进行分解。

> [!note]
>
> **抽象概括**
>
> 我们把不共线的向量 $\vec{e_1}，\vec{e_2}$ 叫作向量 $\vec a$ 的一组**基**，记为 $\{\vec{e_1}，\vec{e_2}\}$
>
> 若基中的两个向量互相垂直，则称这组基为**正交基**，在正交基下向量的线性表示称为**正交分解**；
>
> 若基中的两个向量是互相垂直的单位向量，则称这组基为**标准正交基**。在标准正交基下进行向量分解，许多有关度量的问题就变得较为简单



## 平面向量及运算的坐标表示

- 设点 $A(x_{1},y_{1}),B(x_{2},y_{2})$，则 $\overrightarrow{AB}=(x_2-x_1,y_2-y_1)$
  - 抽象概括：一个向量的坐标等于其终点的坐标减去起点的坐标

- 若 $a=(x_{1},y_{1})，b=(x_{2},y_{2})$，当 $b\neq0$ 时，向量 $a,b$ 共线的充要条件是 $x_1y_2-x_2y_1=0$
  - 记忆方法：外项积 减 内项积
