# 三角形全等

## 答题习惯

- 一定要标注，用不同颜色的笔分组标记等角等边

## 全等三角形构造方法

1. 平移法（主要用于识别全等）
2. 翻折法（轴对称）：主要指角平分线常用辅助线模型
   - 两端引垂线
   - 构对称全等
   - 构等腰
3. 旋转构全等
   - 手拉手模型
   - 鸡扒模型（旋转构造手拉手全等）
   - 一线三等角模型
   - 中线倍长模型
   - 半角模型（一般与截长补短相结合适用）
   - 对角互补模型

## 手拉手模型

### 手拉手

- 模型定义：两等腰三角形，共顶点，顶角相等，有一组三角形是永远全等的。手拉手是共顶点旋转模型的一种，也是重要的相似模型
- 首先找到公共顶点，简称头。然后依次找大图的左手，小图的左手，确认2个左手的连线。再找大图的右手，小图的右手，确认右手的连线。两个连线分别与顶点构成一组全等三角形（简记头左左 全等 头右右）（SAS）
- 左手连线与右手连线形成的夹角，叫做拉手夹角，拉手夹角＝顶角（“8”字形证明）
- 手拉手模型衍生出来的性质（边相等，角相等），可以与其他模型结合使用
- 一个不等边三角形通过同样的方法，做手拉手模型，得到的一组三角形不会全等，会相似
- 总结两点：第一，往往涉及共点旋转的问题，会产生手拉手模型；第二，如果共的这个点对应的两条边，它是相等的，那么得到的手拉手模型产生的三角形，两个是全等的；如果共旋转的顶点，它邻的两个边是不等的，产生的三角形不是全等的，而是相似的



- 构造手拉手模型：等腰三角形手拉手，正方形手拉手等等



- 典型例题：两等边三角形手拉手，八大结论（详见B站）

##  一线三等角模型

- 一条直线上有三个相等的角，或者说有三个相等的角的顶点在同一条直线上，再加上一组边相等，就会产生全等三角形（AAS 或 ASA）（互余倒角或者外角定理倒角）

- 构造一线三垂直：等腰直角三角形、正方形。过直角顶点做直线，往直线做高



- 预学延伸：一线三等角可以构造等腰三角形
