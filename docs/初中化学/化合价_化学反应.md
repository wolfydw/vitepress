## 化合价

### 口诀

> 一价氢氯钾钠银
>
> 二价氧钙钡镁锌
>
> 三铝四硅五价磷
>
> 二三铁二四碳 $FeO$、$Fe_2O_3$、$CO$、$CO_2$
>
> 二四六硫都齐全 $H_2S$、$FeS$、$SO_2$、$H_2SO_4$、$\ce{Fe(OH)2}$、\ce{Fe(OH)2}
>
> 铜贡二价最常见
>
> 单质零价永不变

口诀中是初中化学的化合物中常见的化合价，并不代表该元素没有其他的化合价。例如Cu除了+2价，还有+1价

### 铁的化合价

铁常见：+2、+3

分别对应铁离子$Fe^{3+}$和亚铁离子$Fe^{2+}$，例如2个$OH^-$和1个铁离子结合成$Fe(OH)_2$，通过计算得出铁的化合价为+2，所以读作氢氧化亚铁。

**铁的氧化物**
|名称|化学式|相关知识点|
|:-|:-|:-|
|氧化亚铁|$FeO$|黑色固体|
|氧化铁|$Fe_2O_3$|铁锈的主要成分，红褐色|
|四氧化三铁|$Fe_3O_4$|磁铁，黑色|


1. 铁在纯氧中燃烧可以得到四氧化三铁
   $$
   3Fe+20_{2}\xlongequal{\text{点燃}}Fe_{3}O_{4}
   $$

   > 怎么理解四氧化三铁？
   >
   > 在Fe₃O₄里，铁显两种价态，一个铁原子显+2价，两个铁原子显+3价，所以说四氧化三铁可看成是由FeO与Fe₂O₃组成的化合物，可表示为FeO·Fe₂O₃，而不能说是FeO与Fe₂O₃组成的混合物。

2. 铁的冶炼

$$
Fe_{2}O_{3}+3CO\xlongequal{\text{高温}}2Fe+3CO_{2}
$$

3. 置换反应中铁均为2价的亚铁离子

4. 复分解反应中化合价不变



## 化学方程式

### 什么时候标沉淀和气体

> 前面没有而后面有的，要加标记

$C+2CuO\xlongequal{\text{高温}}2Cu+CO_{2}\uparrow$

~反应前没有气体，反应后有气体，所以要加上标~

$2NaOH+Cu(NO_{3})_{2}=2NaNO_{3}+Cu(OH)_{2}\downarrow$

~反应前没有固体，反应后有沉淀，所以要加下标~

### 配平

Fe  + O₂ + H₂O → Fe(OH)₂

Fe  + O₂ + H₂O → Fe(OH)₃



## 化学反应类型

- 化合反应：多合一

- 分解反应：一变多

- 置换反应：单质 + 化合物 → 另一种单质 + 另一种化合物
  - **金属活动顺序**：钾钙钠镁铝、锌铁锡铅(氢)、铜汞银铂金
  - 远距离先置换
  - 置换反应一定有化合价变化

- 复分解反应：2个化合物互相交换成分，生成2个化合物。
  - 没有化合价变化


### 其他反应

- 氧化反应：物质与氧发生的反应（得到氧的反应）
- 还原反应：物质失去氧的反应
  - 初中阶段常见具有还原性的物质：$C$、$CO$、$H_2$
- 中和反应：酸 + 碱 → 盐 + 水，属于复分解反应的一种，本质是酸中的$H^+$与碱中的$OH^-$结合生成水分子
