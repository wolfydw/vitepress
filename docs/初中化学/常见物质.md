# 常见物质

| 物质名称             | 主要成分       | 用途                         |
| -------------------- | -------------- | ---------------------------- |
| 食盐                 | $\ce{NaCl}$    | 调味、食品防腐               |
| 干冰（固态二氧化碳） | $\ce{CO2}$     | 人工降雨、舞台烟雾效果       |
| 生石灰               | $\ce{CaO}$     | 干燥剂                       |
| 熟石灰、消石灰       | $\ce{Ca(OH)2}$ | 配制澄清石灰水、改良酸性土壤 |
| 大理石、石灰石       | $\ce{CaCO3}$   | 建筑材料                     |
| 黄铜（铜锌合金）     | $\ce{Cu、Zn}$  |                              |
| 盐酸                 | $\ce{HCl}$     | 金属除锈                     |
| 硫酸                 | $\ce{H2SO4}$   |                              |
| 碳酸                 | $\ce{H2CO3}$   | 碳酸饮料，常温下易分解       |
| 硝酸                 | $\ce{HNO3}$    |                              |
| 铁锈、赤铁矿         | $\ce{Fe2O3}$   |                              |
| 磁铁                 | $\ce{Fe3O4}$   |                              |
| 双氧水               | $\ce{H2O2}$    |                              |
| 乙醇、酒精           | $\ce{C2H5OH}$  | 燃料、消毒剂                 |
| 氨气                 | $\ce{NH3}$     | 制氮肥                       |
| 苏打、纯碱           | $\ce{Na2CO3}$  | 玻璃制造、洗涤剂原料         |
| 小苏打               | $\ce{NaHCO3}$  | 烘焙膨松剂                   |
| 苛性钠、火碱、烧碱   | $\ce{NaOH}$    | 造纸、生产肥皂、石油精炼     |
| 草木灰               | $\ce{K2CO3}$   | 钾肥                         |
| 臭氧                 | $\ce{O3}$      |                              |
| 天然气               | $\ce{CH4}$     |                              |
| 水煤气               | $\ce{H2、CO}$  |                              |
