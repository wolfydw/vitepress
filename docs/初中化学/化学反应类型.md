

# 化学反应类型

## 四大基本反应

1. 化合反应：多合一

2. 分解反应：一变多

3. 置换反应：单质 + 化合物 → 另一种单质 + 另一种化合物

   - **金属活动顺序**：钾钙钠镁铝、锌铁锡铅(氢)、铜汞银铂金

   - 远距离先置换

   - 置换反应一定有化合价变化

   - 钾、钙、钠三种金属比较活泼，它们会先跟溶液中的水发生反应置换出氢气

4. 复分解反应：两个化合物互相交换阳离子，生成两个新的化合物
	- 一定要有水、气体、沉淀生成
	- 中和反应优先
	- 没有化合价变化

## 其他反应

- 氧化反应：物质与氧气发生的反应（得到氧的反应）
- 还原反应：物质失去氧的反应
  > 初中阶段常见具有还原性的物质：$\ce{C}$、$\ce{CO}$、$\ce{H2}$
- 中和反应：酸 + 碱 → 盐 + 水，属于复分解反应的一种，本质是酸中的 $\ce{H^+}$ 与碱中的 $\ce{OH^-}$ 结合生成水分子

## 如何判断某物质是沉淀

- 含$\ce{K+}$、$\ce{Na+}$、$\ce{NH4+}$、$\ce{NO3-}$均可溶
- $\ce{Cl-}$：除$\ce{AgCl}$沉淀，其余均可溶
- $\ce{SO4^2-}$：除$\ce{BaSO4}$沉淀，其余均可溶
- $\ce{CO3^2-}$：除$\ce{MgCO3}$微溶，其余均为沉淀
- $\ce{OH-}$：除$\ce{Ba(OH)2}$可溶，其余均为沉淀

> 特例：$\ce{Ca(OH)2}$、$\ce{CaSO4}$、$\ce{Ag2SO4}$微溶，初中阶段微溶几乎不考，不用记忆，只需知道$\ce{Ca(OH)2}$是澄清石灰水的主要成分即可。



## 常见八大沉淀

 $\ce{Mg(OH)2、Al(OH)3、Fe(OH)3、Cu(OH)2、BaSO4、BaCO3、CaCO3、AgCl}$
