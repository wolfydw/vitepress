# 浓缩稀释相关计算

## 稀释

稀释就是往浓溶液中加`水`，使它变稀，稀释前后的溶质质量是不变的，可以得到等式：$稀释前溶质质量=稀释后溶质质量$

**会用到的公式**

$m_{溶质}=m_{溶液}\times溶质质量分数(w)$

$m_{溶质}=m_{溶液}-m_{溶剂}$

## 浓缩

添加溶质使浓度从小变大，叫浓缩

**会用到的公式**

$m_{溶质}=m_{溶液}\times溶质质量分数(w)$、$m_{溶质}=m_{溶液}-m_{溶剂}$

$溶质质量分数(w)=\frac{m_{\text{溶质}}}{m\text{溶液}}\times100\%$

### 例题

一、现有 100g10%的硝酸钾溶液:

(1)若加入 10g水后，溶液的溶质质量分数为（       ）

(2)若加入 10g硝酸钾，且全部溶解，则溶解后的溶质质量分数为（       ）

(3)若把原溶液蒸发掉 20g水后，且无溶质析出，则此溶液的溶质质量分数为（       ）

**解析：**

(1) 设加水后溶液的溶质质量分数为$x$

$100g\times10\%=(100g+10g)\times x$

解得$x=...$

(2)$100g\times 10\%+10g=20g$

$\frac{20g}{100g+10g}\times 100\%=...$

(3)$100g\times 10\%+10g=20g$

$\frac{20g}{100g-20g}\times 100\%=...$
