# 化学方程式

## 什么时候标沉淀和气体

**原则：前面没有而后面有的，要加标记**

$\ce{C + 2CuO \xlongequal{高温} 2Cu+ CO_{2} ^}$ 

> 反应前没有气体，反应后有气体，所以要加上标

$\ce{2NaOH+Cu(NO3)2=2NaNO3+Cu(OH)2 v}$

> 反应前没有固体，反应后有沉淀，所以要加下标



## 方程式配平

### 通用方法

1. 将最复杂的化学式的系数暂定为1
2. 按元素逐个配平（可以设未知数）
3. 有分数就去分母



### 例题

$\ce{Fe + O2\xlongequal{\text{点燃}}Fe3O4}$

$\ce{P + O2\xlongequal{\text{点燃}}P2O5}$

$\ce{KMnO4\xlongequal{\triangle}K2MnO4 + MnO2 + O2 ^}$

$\ce{CH4 + O2\xlongequal{\text{点燃}}CO2 + H2O}$ 

$\ce{C2H5OH + O2 \xlongequal{\text{点燃}}CO2 + H2O}$ 

$\ce{CuFeS2 + O2\xlongequal{\text{高温}} CuO + Fe2O3 + SO2}$

$\ce{Fe2O3 + CO\xlongequal{\text{高温}} Fe + CO2}$ 



## 化学反应类型

### 四大基本反应

- 化合反应：多合一
- 分解反应：一变多
- 置换反应：单质 + 化合物 → 另一种单质 + 另一种化合物
  - **金属活动顺序**：钾钙钠镁铝、锌铁锡铅(氢)、铜汞银铂金
  - 远距离先置换
  - 置换反应一定有化合价变化
  - 钾、钙、钠三种金属比较活泼，它们会先跟溶液中的水发生反应置换出氢气
- 复分解反应：2个化合物互相交换成分，生成2个化合物。
  - 一定要有水、气体、沉淀生成
  - 没有化合价变化


### 其他反应

- 氧化反应：物质与氧气发生的反应（得到氧的反应）
- 还原反应：物质失去氧的反应
  - 初中阶段常见具有还原性的物质：$\ce{C}$、$\ce{CO}$、$\ce{H2}$
- 中和反应：酸 + 碱 → 盐 + 水，属于复分解反应的一种，本质是酸中的$\ce{H^+}$与碱中的$\ce{OH^-}$结合生成水分子



## 如何判断某物质是沉淀

- 含$\ce{K+}$、$\ce{Na+}$、$\ce{NH4+}$、$\ce{NO3-}$均可溶
- $\ce{Cl-}$：除$\ce{AgCl}$沉淀，其余均可溶
- $\ce{SO4^2-}$：除$\ce{BaSO4}$沉淀，其余均可溶
- $\ce{CO3^2-}$：除$\ce{MgCO3}$微溶，其余均为沉淀
- $\ce{OH-}$：除$\ce{Ba(OH)2}$可溶，其余均为沉淀

特例：$\ce{Ca(OH)2}$、$\ce{CaSO4}$、$\ce{Ag2SO4}$微溶，初中阶段微溶几乎不考，不用记忆，只需知道$\ce{Ca(OH)2}$是澄清石灰水的主要成分即可。
