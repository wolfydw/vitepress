# 化学方程式

## 什么时候标沉淀和气体

**原则：前面没有而后面有的，要加标记**

$\ce{C + 2CuO \xlongequal{高温} 2Cu + CO_{2} ^}$ 

> 反应前没有气体，反应后有气体，所以要加上标

$\ce{2NaOH + Cu(NO3)2 \xlongequal{} 2NaNO3 + Cu(OH)2 v}$

> 复分解反应是在溶液中进行，反应前溶液中没有沉淀，反应后有沉淀，所以要加下标



## 方程式配平

### 通用方法

1. 将最复杂的化学式的系数暂定为 1
2. 按元素逐个配平（可以设未知数）
3. 有分数就去分母



### 例题

$\ce{Fe + O2\xlongequal{\text{点燃}}Fe3O4}$

$\ce{P + O2\xlongequal{\text{点燃}}P2O5}$

$\ce{KMnO4\xlongequal{\triangle}K2MnO4 + MnO2 + O2 ^}$

$\ce{CH4 + O2\xlongequal{\text{点燃}}CO2 + H2O}$ 

$\ce{C2H5OH + O2 \xlongequal{\text{点燃}}CO2 + H2O}$ 

$\ce{CuFeS2 + O2\xlongequal{\text{高温}} CuO + Fe2O3 + SO2}$

$\ce{Fe2O3 + CO\xlongequal{\text{高温}} Fe + CO2}$

| 碳不完全燃烧：<br/> $\ce{2C +O2\xlongequal{点燃}2CO}$       |                                                              | 门窗紧闭烧煤炉中毒的原因                                     |
| 硫燃烧：<br/> $\ce{S +O2\xlongequal{点燃}SO2}$              | （空气中）发出微弱的淡蓝色火焰；（氧气中）发出明亮的蓝紫色火焰；生成具有刺激性气味的气体。 | 1. 集气瓶中预留少量水的作用：吸收生成的 $\ce{SO2}$ ，防止污染空气。2. 含硫的矿物燃料燃烧产生的 $\ce{SO2}$ 等会造成酸雨。 |
| 铁丝燃烧：<br/> $\ce{3Fe +2O2\xlongequal{点燃}Fe3O4}$       | 铁丝在空气中红热；在氧气中剧烈燃烧，火星四射，放出大量的热，生成黑色固体。 | 1. 铁丝绕成螺旋状目的：增大铁丝与氧气的接触面积。2. 下端系一根火柴的作用：引燃铁丝。3. 集气瓶内预留少量水的作用：防止高温熔化物溅落，炸裂瓶底。 |

| 镁燃烧：<br/> $\ce{2Mg+O2\xlongequal{点燃}2MgO}$            | 发出耀眼的白光，生成白色固体。                               | 质量守恒定律的探究                                           |
| 一氧化碳燃烧：<br/> $\ce{2CO+O2\xlongequal{点燃}2CO2}$      | 燃烧，火焰呈蓝色                                             | 煤炉里煤层上方的蓝色火焰是一氧化碳在燃烧。                   |
| 二氧化碳与水反应：<br/> $\ce{CO2 +H2O=H2CO3}$               |                                                              | 如何通过对比实验证明二氧化碳与水发生了反应？                 |
| 铝和氧气反应：<br/> $\ce{4Al +3O2=2Al2O3}$                  |                                                              | 铝具有很好抗腐蚀性原因：铝在空气中与氧气反应，表面生成一层致密的氧化铝薄膜，阻止铝进一步被氧化。 |
| 生石灰与水反应：<br/>$\ce{CaO +H2O=Ca(OH)2}$                | 反应中放出大量热。                                           | 生石灰具有吸水性，可用作干燥剂。                             |
| 碳和二氧化碳反应：<br/> $\ce{C +CO2\xlongequal{高温}2CO}$   |                                                              | 反应吸热                                                     |
| $\ce{2Hg +O2\xlongequal\triangle 2HgO}$                     | 产生红色固体                                                 | 拉瓦锡利用该反应测定空气中氧气含量。                         |
