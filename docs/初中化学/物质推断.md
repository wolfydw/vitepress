# 物质推断题的思路

## 常见物质

推断题给出的"常见物质”可分为以下六大类：

1. 单质：$\ce{O2}$、$\ce{H2}$、$\ce{C}$、$\ce{Fe}$、$\ce{Cu}$
2. 氧化物：$\ce{CO2}$、$\ce{CO}$、$\ce{H2O}$、$\ce{H2O2}$、$\ce{CuO}$、$\ce{CaO}$、$\ce{Fe2O3}$、$\ce{Fe3O4}$
3. 酸：$\ce{HCl}$、$\ce{H2SO4}$
4. 碱：$\ce{NaOH}$、$\ce{Ca(OH)2}$
5. 盐：$\ce{Na2CO3}$、$\ce{CaCO3}$、$\ce{NaHCO3}$、$\ce{CuCl2}$、$\ce{CuSO4}$、$\ce{NaCl}$
6. 有机物：$\ce{ CH4}$、$\ce{C2H5OH}$



## 沉淀

- 含$\ce{K+}$、$\ce{Na+}$、$\ce{NH4+}$、$\ce{NO3-}$均可溶
- $\ce{Cl-}$：除$\ce{AgCl}$沉淀，其余均可溶
- $\ce{SO4^2-}$：除$\ce{BaSO4}$沉淀，其余均可溶
- $\ce{CO3^2-}$：除$\ce{MgCO3}$微溶，其余均为沉淀
- $\ce{OH-}$：除$\ce{Ba(OH)2}$可溶，其余均为沉淀

特例：$\ce{Ca(OH)2}$、$\ce{CaSO4}$、$\ce{Ag2SO4}$微溶，初中阶段微溶几乎不考，不用记忆，只需知道$\ce{Ca(OH)2}$是澄清石灰水的主要成分即可。



## 特殊颜色

|状 态|颜 色|物 质 或 离 子|
| :- | :- | :- |
|固体|黑 色|$\ce{C}$、$\ce{Fe}$（铁粉）、$\ce{CuO}$、$\ce{Mn2O}$、$\ce{Fe3O4}$|
||红 色|$\ce{Cu}$、$\ce{P}$（红磷）、$\ce{HgO}$、$\ce{Fe2O3}$（铁锈）|
||暗 紫 色|$\ce{KMnO4}$、$\ce{I2}$|
||淡 黄 色|$\ce{S}$（硫粉）|
|沉 淀|蓝 色|$\ce{Cu(OH)2}$|
||红 褐 色|$\ce{Fe(OH)3}$|
||白 色|其余沉淀大部分都是白色|
|溶液|蓝 色|$\ce{Cu^2+}$|
||浅 绿 色|$\ce{Fe^2+}$|
||黄 色|$\ce{Fe^3+}$|
|火焰|蓝 紫 色|$\ce{S}$在$\ce{O2}$中燃烧|
||淡 蓝 色|$\ce{S}$、$\ce{H2}$在空气中燃烧|
||蓝 色|$\ce{CO}$、$\ce{CH4}$在空气中燃烧|
||苍 白 色|$\ce{H2}$在$\ce{Cl2}$中燃烧|



## 溶于水放热 & 吸热

$\ce{Na0H}$、$\ce{CaO}$、浓$\ce{H2SO4}$溶于水**放热**

$\ce{NH4NO3}$溶于水**吸热**
