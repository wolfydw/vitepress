# 物质推断题的思路

## 常见物质

推断题给出的"常见物质”可分为以下六大类别：

1. 单质：$\ce{O2}$、$\ce{H2}$、$\ce{C}$、$\ce{Fe}$、$\ce{Cu}$
2. 氧化物：$\ce{CO2}$、$\ce{CO}$、$\ce{H2O}$、$\ce{H2O2}$、$\ce{CuO}$、$\ce{CaO}$、$\ce{Fe2O3}$、$\ce{Fe3O4}$
3. 酸：$\ce{H2SO4}$、$\ce{HCl}$、$\ce{HNO3}$
4. 碱：$\ce{NaOH}$、$\ce{Ca(OH)2}$
5. 盐：$\ce{Na2CO3}$、$\ce{CaCO3}$、$\ce{NaHCO3}$、$\ce{CuCl2}$、$\ce{CuSO4}$、$\ce{NaCl}$、$\ce{AgNO3}$ 
6. 有机物：$\ce{ CH4}$、$\ce{C2H5OH}$

要非常熟悉这些常见物质的相关知识点

## 物质用途

- 冶炼金属——$\ce{CO、H2、C }$
- 食品防腐——氮气$\ce{N2}$ 
- 食品干燥剂——生石灰 $\ce{CaO}$
- 食品脱氧剂—— 铁粉$\ce{Fe}$
- 建筑材料——大理石、石灰石$\ce{CaCO3}$ 
- 液体燃料——乙醇$\ce{C2H5OH}$ 
- 改良酸性土壤——$\ce{Ca(OH)2}$ 
- 配制波尔多液——$\ce{CuSO4}$ 、$\ce{Ca(OH)2}$ 
- 胃液成分、洁厕灵、除铁锈——盐酸$\ce{HCl}$ 
- 铅蓄电池、气体干燥剂——硫酸$\ce{H2SO4}$ 
- 调味品、融雪剂——氯化钠$\ce{NaCl}$ 
- 发酵粉——碳酸氢钠$\ce{NaHCO3}$ 
- 治疗胃酸——碳酸氢钠$\ce{NaHCO3}$ 、氢氧化铝$\ce{Al(OH)3}$ 
- 玻璃、洗涤剂、造纸、纺织——碳酸钠$\ce{Na2CO3}$ 
- 肥皂、清洁油污——氢氧化钠$\ce{NaOH}$ 
- 常见的复合肥——硝酸钾$\ce{KNO3}$ 



## 物质组成特征

- 氧化物中一定含有氧元素
- 具有相同元素组成的气体是 $\ce{CO2}$和 $\ce{CO}$
- 具有相同元素组成的液体是 $\ce{H2O2}$和 $\ce{H2O}$
- 具有相同元素组成的固体是 $\ce{FeO、Fe2O3、Fe3O4}$



## 以反应条件为突破口

由于初中化学中所涉及的化学反应不多，有些化学反应条件特别容易暴露，例如：

- 点燃：（有$\ce{O2}$参加的反应）

- 通电：（电解水）$\ce{2H2O\xlongequal{通电}2H2 ^ + O2 ^}$ 

- 催化剂：$\ce{2KClO3\xlongequal{MnO2} 2KCl + 3O2 ^}$，$\ce{2H2O2\xlongequal{MnO2} 2H2O + O2 ^}$ 

- 高温：

  - 碳酸钙分解：$\ce{CaCO3\xlongequal{高温} CaO + CO2 ^}$ 

  - 其余都是有$\ce{C、CO、H2}$ 参与的还原反应
    $\ce{CO2 + C\xlongequal{高温} 2CO}$ 
    $\ce{2CuO + C\xlongequal{\triangle} 2Cu + CO2 ^}$ 

    $\ce{CuO + CO \xlongequal{\triangle} Cu + CO2 ^}$ 

    $\ce{CuO + H2 \xlongequal{高温} Cu + H2O}$ 
    $\ce{2Fe2O3 + 3C\xlongequal{高温} 4Fe + 3CO2 ^}$ 

    $\ce{Fe2O3 + 3CO\xlongequal{高温} 2Fe + 3CO2 ^}$ 

    $\ce{Fe2O3 + 3H2\xlongequal{高温} 2Fe + 3H2O}$ 

    $\ce{Fe3O4 + 2C\xlongequal{高温} 3Fe + 2CO2 ^}$ 

    $\ce{Fe3O4 + 4CO\xlongequal{高温} 3Fe + 4CO2 ^}$ 

    $\ce{Fe3O4 + 4H2\xlongequal{高温} 3Fe + 4H2O}$ 



## 生活中的化学之最

- 地壳中元素含量顺序：氧、硅、铝、铁
- 人体内含量最多的金属元素：钙
- 人体内含量最多的非金属元素：氧
- 人体内含量最多的物质：水
- 空气中含量最多的元素：$\ce{N}$ 
- 自然界中硬度最大的：金刚石
- 生活中最常用/世界年产量最高的金属：铁
- 导电性、导热性最好的金属：银



## 溶于水放热 & 吸热

溶于水**放热**：氢氧化钠$\ce{NaOH}$、氧化钙$\ce{CaO}$、浓硫酸$\ce{H2SO4}$

溶于水**吸热**：硝酸铵$\ce{NH4NO3}$



## 强酸制弱酸

初中阶段常见的酸，按酸性强弱排序：$\ce{H2SO4}$ > $\ce{HCl}$ > $\ce{HNO3}$

$\ce{H2SO4 + BaCl2\xlongequal{}2HCI + BaSO4 v}$ 

$\ce{HCl + AgNO3\xlongequal{}HNO3 + AgCI v}$ 

$\ce{H2SO4 + Ba(NO3)2\xlongequal{}2HNO3 + BaSO4 v}$ 

总结：强酸 + 金属盐 == 弱酸 + 沉淀



## 反应规律

**能与金属氧化物反应的**：

- 金属氧化物 + 酸 → 金属盐 + $\ce{H2O}$
- 金属氧化物 + 还原性物质 → 金属单质 + $\ce{CO2}$ 或 $\ce{H2O}$

**能与$\ce{CO2}$反应的**：

- $\ce{CO2}$ + 碱 → 碳酸某 + $\ce{H2O}$ 

- $\ce{CO2 + C\xlongequal{高温} 2CO}$
